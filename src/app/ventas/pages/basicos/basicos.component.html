Pipes Fechas

Pipes incluidos en Angular - Para manejar el formato de la fecha Más Información

Se coloca la fecha recibida de un ' new Date ' seguido de ' | ' y por ultimo escribimos el comando ' date ' :

{{ ejemploDate }}

Con lo que conviete ' {{ fecha }} ' en el formato siguiente:

{{ fecha | date }}

Para formatear la fecha a nuestro gusto podemos utilizar por ejemplo " date:'short' "{{ ejemploDateSort }} :

{{ fecha | date:'short' }}

O bien utilizamos " 'dd / MM / yyyy, hh:mm a' " en formato manual {{ ejemploDateSort2 }} :

{{ fecha | date:'dd / MM / yyyy, hh:mm a' }}

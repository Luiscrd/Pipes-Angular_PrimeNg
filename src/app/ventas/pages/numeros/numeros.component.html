DecimalPipe

Pipes incluidos en Angular - Para manejar decimales y monedas Más Información

Se coloca el caracter ' | ' seguido del comando ' number ' :

{{ ejemploNumero }}

Para que coloque puntos al numero :

{{ ventasNetas | number }}

Se coloca " '1.2-2' " seguido de ' : ' y el comando ' number ' :

{{ ejemploNumeroDecimal }}

Para que muestre solo 2 decimales :

{{ ventasNetas | number:'1.2-2' }}

Se coloca ' € ' seguido de ' : ' y el comando ' currency ' :

{{ ejemploNumeroMoneda }}

Para que lo muestre como moneda :

{{ ventasNetas | currency:'€' }}

PercentPipe

Pipes incluidos en Angular - Para manejar porcentajes Más Información

Se coloca el caracter ' | ' seguido del comando ' percent ' :

{{ ejemploNumeroPorcentaje }}

Para darle un formato de porcentaje redondeandolo :

{{ porcentaje | percent }}

Se coloca " '1.2-2' " seguido de ' : ' y el comando ' percent ' :

{{ ejemploNumeroPorcentaje2 }}

Para que muestre los 2 decimales sin hacer redondeo :

{{ porcentaje | percent:'1.2-2' }}

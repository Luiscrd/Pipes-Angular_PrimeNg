Este Pipe se utiliza para obtenr una lista coratada a nuestro antojo:

Estructura:

{{ listaSpliceEjemplo }}

Ejemplos:

{{ listaSpiceEjempo1 }}

{{ listaSpiceEjempo2 }}

{{ listaSpiceEjempo3 }}

Resultados:

La lista recortada del primero( 0 ) al segundo ( 2 - no incluido ) es: ' {{ listaSplice | slice:0:2 }} '

La lista recortada del segundo( 1 ) al tercero ( 3 - no incluido ) es: ' {{ listaSplice | slice:1:3 }} '

La lista recortada del tercero( 3 ) al ultimo ( listaSplice.length+1 ) es: ' {{ listaSplice | slice:3:listaSplice.length+1 }} '
